# Web Penetration Testing Learning Lab

This repository contains a penetration testing report created as part of a personal web security learning project conducted in a controlled lab environment.

The assessment focuses on identifying and exploiting common web application vulnerabilities using intentionally vulnerable systems.

## Methodology

OWASP Web Security Testing Guide (WSTG)

Testing approach:

* Black Box Testing
* Gray Box Testing

## Vulnerabilities Identified

High Severity:

* Stored Cross-Site Scripting (Session Hijacking)
* Insecure Direct Object Reference (IDOR)
* Weak Password Hashing
* Session Cookie Forgery (JWT Manipulation)

Medium Severity:

* Reflected XSS
* Outdated Components

## Tools Used

* Browser Developer Tools
* Wappalyzer
* sqlite3
* webhook.site
* CrackStation

## Lab Platforms

* TryHackMe
* OWASP Juice Shop

## Disclaimer

All testing activities were performed in intentionally vulnerable training environments for educational purposes only.
